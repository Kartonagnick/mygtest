[![P]][M] component v0.0.1
==========================
Определяет макросы:  
  - `dDEATH_TEST`         - Смертельный тест. В релизе сворачиваетя в пустоту.  
  - `ASSERT_DEATH_DEBUG`  - Смертельный тест. В релизе сворачиваетя в пустоту.  
  - `SKIP_TEST_GENERATOR` - Пропустить тест-генератор.  
  - `SKIP_STRESS_TEST`    - Пропустить стресс-тест.  
  - `TEST_COMPONENT`      - Объявляет тест.  
  - `_TEST_COMPONENT`     - Игнорировать тест.  
  - `_TEST`               - Игнорировать тест (старый стиль).  

Если макрос не был объявлен: `dDISABLE_DPRINT`,  
тогда `component.hpp` подключает `dprint.hpp`
