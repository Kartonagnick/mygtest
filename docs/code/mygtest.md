[![S]][M] mygtest v0.0.1
========================
Главный заголовок `mygtest.hpp` подключает оригинальные заголовки `gtest`:  
  - если объявлен макрос `dGMOCK_LIBRARY_USED_`  
    тогда подключается `gmock`  
  - если объявлен макрос `dGTEST_LIBRARY_USED_`  
    тогда подключается `gtest`  
  - по умолчанию подключается `gmock`  

Макросы `dGTEST_LIBRARY_USED_` или `dGMOCK_LIBRARY_USED_`  
должны определяться на уровне системы сборки.
